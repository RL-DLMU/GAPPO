# 自适应交通信号控制
本项目使用多智能体强化学习使自适应交通信号控制中的不同路口间的通信摆脱对实时通信的依赖。以下内容介绍了项目的环境要求以及使用方式。

## 环境要求：
- 操作系统：Windows/Linux/MacOS（已在Windows 以及Linux 上测试）。
- Python版本：推荐使用Python 3.7。
- 主要依赖包括：
  - PyTorch
  - NumPy
  - Matplotlib
  - Math
## 使用方式：
## 训练模型

- `python GAPPO/run.py`
